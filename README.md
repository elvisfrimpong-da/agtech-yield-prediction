# 🌾 AgTech Crop Yield Intelligence

An interactive machine learning web application that predicts crop yield based on environmental and agricultural factors. Built with Python, Scikit-learn, and Streamlit, this project demonstrates how data-driven insights can support agricultural decision-making.

---

## 🚀 Live Application

**Try the app here:**
https://agtech-yield-prediction.streamlit.app/

---

## 📌 Features

* Predict crop yield using a trained Multiple Linear Regression model
* Interactive dashboard with tab-based navigation (Prediction, Insights, About)
* Environmental insights based on user inputs
* Country and crop selection
* Real-time predictions
* Mobile-friendly and responsive design

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Git & GitHub

---

## 📊 Model Performance

The regression model achieved:

* **R² Score:** 0.834

This indicates that the model explains approximately **83.4% of the variance** in crop yield based on the selected input features.

---

## 📊 Model Information

The model was trained on a global agricultural dataset containing:

* Crop type
* Country/Area
* Average rainfall
* Average temperature
* Pesticide usage
* Historical crop yield data

### Data preprocessing steps:

* Log transformation to handle skewed variables
* One-hot encoding of categorical variables (Area, Item)
* Feature selection and model fitting using Multiple Linear Regression

---

## 🎯 Objective

To demonstrate the application of machine learning in agriculture by developing a practical tool that estimates crop yield from environmental conditions and agricultural inputs.
